# Nom Population Superficie

Aide

Mode Jeu

  • Trouvez le pays demandé en cliquant sur la carte
  • 3 vies — une erreur ou un timeout coûte une vie
  • Répondez vite pour gagner plus de points (max 1000)
  • Les questions ratées sont reposées 2 tours plus tard
  • La partie se termine quand tous les pays sont trouvés ou quand les vies sont épuisées

Mode Apprentissage

  • Cliquez sur un pays sur la carte pour voir sa capitale et sa population
  • Cliquez sur un pays dans la liste pour le localiser sur la carte
  • Triez la liste par nom, population ou superficie

Navigation carte

  • Molette : zoom
  • Clic + glisser : déplacer la carte
  • Double-clic : réinitialiser le zoom
